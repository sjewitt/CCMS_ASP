Class Content

Summary

Constructor

  • Content(Integer contentId)

Instance Properties

  • authGroup Integer indicating authorisation group.
  • createdDate Date of creation.
  • createdUser Integer created user ID.
  • id Integer Content ID
  • updatedDate Date of last update (corresponds to associated ContentInstance createdDate)
  • updatedUser Integer updated user ID

Instance Methods

  • getInstanceList() Returns an array of ContentInstance objects associated with this Content. A valid Content ID must be used to construct the instance.
  • getInstance([instanceId]) Return the specified ContentInstance.
  • getActiveInstance() Return the most recent ACTIVE instance.
  • getCreatedUser() Return the Content created User.
  • getUpdatedUser() Return the Content updated User.
