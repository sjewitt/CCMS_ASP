Class Viewtree

Summary

Utility class to manage viewtree nodes and hierarchy.

Constructor

  • Viewtree()

Instance Methods

  • getNode(Integer ViewtreeId) - Retrieve specified Node object.
  • getRoot() - Retrieve root Node object.
  • getLevel(Node node) - Retrieve Integer hierarchical level for specified Node
  • getVTRefs(Page page) - Retrieve an array of Nodes which reference supplied Page object.
  • getVTRefCount(Page page) - Retrieve Integer number of viewtree references to supplied Page object.
  • addNode(Node parentNode,Node newNode) - Editing method. Insert specified new Node as child of specified parent Node.
