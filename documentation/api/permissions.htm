Class Permissions

Summary

Binary permission constants.

Class Properties

  • ADMINISTRATOR - all rights.
  • CHANGESTATE - set content to active right.
  • CREATEPAGE - Create page right.
  • CREATECONTENT - Create content right.
  • EDITPAGE - add/remove from slots, page core props.
  • EDITCONTENT - Edit content right.
  • DELETEPAGE - Delete page right.
  • DELETECONTENT - Delete content right.
  • BROWSEONLY - no edit rights, just logged in.
  • EDITVIEWTREE - edit node position in viewtree.
  • UPLOADBINARY - Upload binary right.
