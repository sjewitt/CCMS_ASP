Class ContentMap

Summary

Used internally. Instances of this class map Page to Content to position on layout. Instantiation with a valid Page ID generates an array of objects mapping Content ID to Slot ID:

mapping[n] = { slotId = n; contentId = n; }

Integer slotId is ultimately parsed from 'n' in the CCMS tags {CMS_CONTENT_n} (see XXXXXXXX content storing TODO)

Integer contentId is the associated Content ID

Constructor

  • ContentMap(Integer PageId)

Instance Properties

  • pageId - Integer Page ID
  • mapping - Array of objects. This Array is used by the callee to perform further actions.
