Class Node

Summary

Abstraction of a position in the content hierarchy. Requires a valid Node ID in the constructor.

Constructor

  • Node(Integer NodeID)

Instance Properties

  • id - Integer Node ID.
  • parentId - Integer Direct hierarchical ancestor Node ID. Zero if root node.
  • pageId - Integer ID of referenced Page object.
  • hasChildren - Boolean does this Node have direct hierarchical descendents?
  • level - Integer hierarchical level, counting from root level as zero.
  • layoutId - Integer ID of Layout object corresponding to this hierarchical position. Note, this property may be inherited from a direct ancestor node.
  • layoutInherited - Boolean indicates whether the layout ID is inherited (TRUE) or is a direct property of this Node.
  • isCurrentNode - Boolean. Specifically in the context of an array of Nodes and navigation logic: this value is TRUE when the node in an array of nodes is the one being rendered to the browser. Other nodes isCurrentNode property in the array have a FALSE value.

Instance Methods

  • getChildren - Retrieve an array of Nodes represending the direct descendents of the current Node.
  • getParent - Retrieve the Node direct parent of the current Node. TODO: Undefined if current Node is the root node.
  • getPage - Retrieve the Page object referenced by this Node.
