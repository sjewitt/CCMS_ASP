Constructor

Instance Properties

  • id - Integer ID of current Page object
  • exists - Boolean does page specified by ID in the constructor actually exist?
  • linkText(*) - String short text describing content. Default use is for text of HREFs.
  • title(*) - String short text describing content. Default use is for the browser window title.
  • description(*) - String short text describing content. Default use is for HTML metatext DESCRIPTION tag value.
  • keywords(*) - String short text describing content. Default use is as comma-separated word list as HTML metatext KEYWORDS tag value.
  • content - Array of referenced Content objects. Populated by getContent() method, described below.
  • createdUser - Integer created user ID.
  • layoutId - Integer Layout ID.

Instance Methods

  • getContent - Populate this object's content array.
  • update - Editing method. Update the core properties(*) of this Page object.
